PREDICTING LEAGUE OF LEGENDS MATCH OUTCOMES FROM EARLY GAME STATISTICS

Download Full Report View on GitHub

My name is Aaron Wen and I'm currently a senior at University of California, Berkeley studying Data Science & Physics. This project was created for the final project of Physics C111: Introduction to Machine Learning in Physics.

Key Features & Methodology

Feature Importance Analysis

Identified Gold Difference, Experience Difference, and Objective Control (Dragons/Heralds) as the most critical early-game factors driving victory.

Model Comparison

Evaluated 6 models (Logistic Regression, SVM, Random Forest, etc.). Logistic Regression with L1 regularization performed best (AUC 0.80).

Strategic Insights

Quantified the impact of objectives: securing both the first Dragon and Herald leads to a 73% win probability.

About the Project

Leveraging machine learning to understand the "snowball effect" in competitive gaming.

This project analyzes nearly 10,000 Diamond-tier League of Legends matches to determine how early-game advantages (first 10 minutes) translate into match victories. By using only early-game data, we simulate a real-time predictive scenario, providing insights into the most impactful strategic decisions players can make in the laning phase.
